## 1. Описание проекта

Веб-приложение для проведения интеллектуальных викторин в реальном времени. Пользователи создают игровые комнаты, приглашают друзей и соревнуются в знаниях. Игра проходит по раундам: игроки видят вопрос, выбирают ответ из вариантов, получают очки в зависимости от правильности и скорости реакции. По окончании игры результаты сохраняются в глобальный лидерборд.

### Основная цель
Реализовать стабильное real-time взаимодействие между игроками с справедливой системой подсчета очков и синхронизацией игрового процесса.

## 3. Функциональные требования

### 3.1. Модуль авторизации и профиля
- Регистрация по email/паролю (или nickname + пароль для простоты)
- Вход в систему
- Профиль пользователя: аватар (генерация по умолчанию), никнейм, дата регистрации
- **Личная статистика:**
  - Всего игр сыграно
  - Побед
  - Процент правильных ответов
  - Общий рейтинг (сумма очков за все игры)

### 3.2. Модуль комнат (Лобби)
**Создание комнаты:**
- Название комнаты
- Максимальное количество игроков (2-8)
- Количество вопросов (5, 10, 15)
- Время на ответ (10, 20, 30 секунд)

**Список комнат:**
- Отображение всех комнат
- Поиск по коду

**Лобби:**
- Список игроков с аватарками
- Индикаторы готовности (флажок "Готов")
- Кнопка "Начать игру" (активна только для Host, когда минимум 2 игрока готовы)

### 3.3. Игровой процесс (Real-time)
#### Этапы игры:
1. **Обратный отсчет:** После нажатия "Старт" 3-секундный отсчет для всех.
2. **Раунд:**
   - **Вопрос:**
     - Текст вопроса отображается на экране
     - несколько вариантов ответа
     - Таймер обратного отсчета (визуальный прогресс-бар)
   - **Прием ответов:**
     - Игрок выбирает вариант → кнопки блокируются
     - Ответ отправляется на сервер с временной меткой
   - **Обработка:**
     - Сервер проверяет правильность
     - Начисление очков (формула ниже)
   - **Результаты раунда:**
     - Всем показывается правильный ответ
     - Таблица с очками за текущий раунд (всплывающее окно на 3 сек)
3. **Завершение игры:**
   - Финальная таблица результатов (пьедестал)
   - Кнопка: "На главную"

#### Формула подсчета очков:
Базовые очки = 100
Если ответ правильный:
Очки = Базовые очки + (Оставшееся время / Максимальное время) * 50 (бонус за скорость)
Если ответ неправильный:
Очки = 0
### 3.4. Лидерборд
- Глобальная таблица рейтинга (Топ-100)
- **Сортировка:**
  - По сумме очков за все время
  - По количеству побед
- Ежемесячный/еженедельный рейтинг (опционально)

### 3.5. База вопросов
**Структура вопроса:**
- Текст вопроса
- несколько вариантов ответа
- Индекс правильного ответа (от 0 до n-1)

**Наполнение:** Минимум 100 вопросов при старте

// questions.js
export function getRandomQuestions(count) {
  const shuffled = [...QUESTIONS_DB];
  // Алгоритм Фишера-Йетса для перемешивания
  for (let i = shuffled.length - 1; i > 0; i--) {
    const j = Math.floor(Math.random() * (i + 1));
    [shuffled[i], shuffled[j]] = [shuffled[j], shuffled[i]];
  }
  return shuffled.slice(0, Math.min(count, shuffled.length));
}

    const decoded = jwt.verify(token, process.env.JWT_SECRET);

    next();

    res.status(401).json({ error: 'Неверный токен' });

  const question = room.selectedQuestions[room.currentQuestion];
  
  const isCorrect = answerIndex === question.correctAnswer;
  const timeLeft = Math.max(0, room.settings.timeLimit - timeSpent);

    isCorrect,
    pointsEarned: points,
    correctAnswer: question.correctAnswer
  });
});

## 7. WebSocket Events (Socket.io)

| **Пространство имен** | `/game` |
|----------------------|---------|

### Клиент → Сервер

| **Событие** | **Описание** | **Данные** |
|-------------|--------------|------------|
| `join_room` | Подключение к комнате | `{ roomId: "uuid", userId: "uuid" }` |
| `player_ready` | Изменение статуса готовности | `{ roomId: "uuid", isReady: boolean }` |
| `start_game` | Запуск игры (только Host) | `{ roomId: "uuid" }` |
| `submit_answer` | Отправка ответа на вопрос | `{ roomId: "uuid", questionId: number, answerIndex: number, timeSpent: number }` |

### Сервер → Клиент

| **Событие** | **Описание** | **Данные** |
|-------------|--------------|------------|
| `room_state` | Обновление состава комнаты | `{ players: Array, settings: Object }` |
| `player_joined` | Игрок присоединился | `{ user: Object, players: Array }` |
| `player_left` | Игрок покинул | `{ userId: "uuid", players: Array }` |
| `player_ready_update` | Статус готовности обновлен | `{ userId: "uuid", isReady: boolean }` |
| `game_starting` | Начало игры (обратный отсчет) | `{ countdown: number }` |
| `new_question` | Новый вопрос | `{ questionId: number, text: string, options: string[], timeLimit: number, currentQuestion: number, totalQuestions: number }` |
| `answer_result` | Результат ответа (индивидуально) | `{ isCorrect: boolean, pointsEarned: number, correctAnswer: number }` |
| `game_over` | Игра окончена | `{ results: Array, winner: Object }` |
